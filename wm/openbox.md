# Openbox

## Creating a Theme

Create a dir structure:

* MyTheme
  + openbox-3
    - themerc
    - \*.xbm

* Edit .xbm files in GIMP

Use `obconf`:

* *Create a theme archive* == (.obt)...\*
  + Select the *MyTheme* folder from above
  + Select *Install a new theme...*
    - Navigate to the obt's dir & select that
